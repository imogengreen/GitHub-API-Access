# GitHub-API-Access
Part of the coursework for CSU33012 Software Engineering.

# Requirements
**1. Python**

**2. Pip:** This can be installed by running the _get-pip.py_ file that is located within the repository.

**3. PyGithub:** Command for installation: _pip install PyGithub_ 

**4. Pymongo:** Command for installation: _pip install Pymongo_

**5. JSON** (which is a Python module and can simply be imported)

**6. Flask**
