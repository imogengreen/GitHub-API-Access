# GitHub-API-Access
Part of the coursework for CSU33012 Software Engineering.

# Requirements
1. Python
2. Pip 
    This can be installed by running the get-pip.py file that is located within the repository.
3. PyGithub
    Command for installation:
    pip install PyGithub 
4. Pymongo
    Command for installation:
    pip install Pymongo
5. JSON (which is a Python module and can simply be imported)
